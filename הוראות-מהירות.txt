🚀 התקנה ב-3 שלבים:

┌─────────────────────────────────────────────────────┐
│ שלב 1: הורד את הקבצים                              │
└─────────────────────────────────────────────────────┘

   git clone https://github.com/noambars121/CursorBeam.git
   cd CursorBeam


┌─────────────────────────────────────────────────────┐
│ שלב 2: הרץ התקנה                                   │
└─────────────────────────────────────────────────────┘

   לחץ פעמיים על: התקן.bat
   
   או:
   
   powershell -ExecutionPolicy Bypass -File install.ps1


┌─────────────────────────────────────────────────────┐
│ שלב 3: מלא פרטים בחלון                             │
└─────────────────────────────────────────────────────┘

   ✓ בחר סיסמה (או צור אוטומטית)
   ✓ אשר את מיקום Cursor (מזוהה אוטומטית)
   ✓ סמן "התקן Tailscale" (מומלץ!)
   ✓ סמן "הפעלה אוטומטית"
   ✓ לחץ "התקן עכשיו"


═══════════════════════════════════════════════════════
    📱 התחברות מהטלפון
═══════════════════════════════════════════════════════

1️⃣ הפעל את השרת:
   לחץ פעמיים על: התחל.bat
   
   (מעכשיו יתחיל אוטומטית עם Windows!)

2️⃣ פתח בטלפון:

   🏠 אותו WiFi:
   http://192.168.1.XXX:9800
   (ההתקנה תראה לך את ה-IP המדויק)

   🌐 Tailscale (מכל מקום):
   http://100.64.X.X:9800
   (הרץ: tailscale ip -4 לגלות את ה-IP)

3️⃣ התחבר:
   הזן את הסיסמה שבחרת בהתקנה

🔥 בעיות נפוצות:

   לא מתחבר מהטלפון?
   → הרץ כמנהל:
   New-NetFirewallRule -DisplayName "Cursor Mobile" `
     -Direction Inbound -LocalPort 9800 -Protocol TCP -Action Allow

   שכחת סיסמה?
   → פתח .env בנוטפד, שנה V2_PASSWORD, הפעל מחדש

   השרת לא מתחיל?
   → netstat -ano | findstr "9800"
   → taskkill /F /PID <מספר>
