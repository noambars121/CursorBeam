# 🚀 מדריך התחלה מהירה - Cursor Mobile

## התקנה ב-3 שלבים פשוטים

### שלב 1️⃣: הורד את הקבצים

```powershell
git clone https://github.com/noambars121/CursorBeam.git
cd CursorBeam
```

### שלב 2️⃣: הרץ את ההתקנה

```powershell
powershell -ExecutionPolicy Bypass -File install.ps1
```

**זהו!** תיפתח חלונית התקנה גרפית.

### שלב 3️⃣: מלא פרטים בחלון ההתקנה

החלון יבקש ממך:

1. **סיסמה** - בחר סיסמה (או לחץ "צור אוטומטית")
2. **מיקום Cursor** - יזוהה אוטומטית (או בחר ידנית)
3. **Tailscale** - סמן אם תרצה חיבור מרחוק מאובטח
4. **הפעלה אוטומטית** - סמן כדי שהשרת יפעל עם Windows

לחץ **"התקן עכשיו"** ותן למערכת לעשות הכל!

## 📱 איך מתחברים מהטלפון?

אחרי ההתקנה, החלון יראה לך את הכתובות:

### אפשרות 1: רשת מקומית (WiFi)
```
http://192.168.1.XXX:9800
```
- פתח את הכתובת בדפדפן בטלפון
- הזן את הסיסמה שבחרת
- זהו!

### אפשרות 2: Tailscale (מומלץ!)
```
http://100.64.X.X:9800
```
- התקן Tailscale גם בטלפון (חינמי!)
- התחבר לאותו חשבון Tailscale
- השתמש בכתובת Tailscale שהמערכת הראתה

## 📲 התקנת PWA (ממש כמו אפליקציה!)

### iPhone/iPad:
1. פתח את הכתובת בדפדפן Safari
2. לחץ על כפתור השיתוף (⬆️)
3. גלול למטה ובחר **"Add to Home Screen"**
4. לחץ "Add"
5. עכשיו יש לך אייקון על המסך הראשי!

### Android:
1. פתח את הכתובת בדפדפן Chrome
2. לחץ על תפריט (⋮)
3. בחר **"Add to Home screen"**
4. לחץ "Add"
5. האפליקציה מותקנת!

## 🔧 הגדרת Tailscale (חיבור מרחוק)

אם בחרת להתקין Tailscale:

### במחשב (Windows):
1. המערכת כבר התקינה את Tailscale
2. פתח את Tailscale מהסרגל (system tray)
3. לחץ "Log in" והתחבר עם חשבון Google/Microsoft/GitHub
4. זהו! אתה מחובר

### בטלפון:
1. הורד את Tailscale מה-App Store / Play Store (חינמי!)
2. פתח את האפליקציה
3. התחבר עם **אותו חשבון** שהשתמשת במחשב
4. עכשיו יש לך גישה מאובטחת מכל מקום בעולם!

## 💡 שימושים נפוצים

### שליחת הודעה
פשוט כתוב בשדה הטקסט ולחץ שלח - בדיוק כמו ב-Cursor!

### אישור פעולות (Tools)
כשהבינה המלאכותית רוצה להריץ פקודה, תראה כפתור "אשר" - פשוט לחץ עליו

### עריכת הודעה
לחץ ארוך (long-press) על הודעה שלך כדי לערוך ולהתחיל שיחה חדשה

### מעבר בין פרויקטים
לחץ על כפתור הפרויקט בראש המסך

### טרמינל
לחץ על כרטיסיית Terminal למטה ותוכל להריץ פקודות

### טעינת היסטוריה
לחץ "טען כל ההיסטוריה" כדי לראות את כל השיחה (לא רק ההתחלה)

## 🆘 בעיות נפוצות ופתרונות

### 🔴 לא מצליח להתחבר מהטלפון

**פתרון 1:** בדוק Firewall
```powershell
# פתח PowerShell כמנהל והרץ:
New-NetFirewallRule -DisplayName "Cursor Mobile" -Direction Inbound -LocalPort 9800 -Protocol TCP -Action Allow
```

**פתרון 2:** בדוק שאתה באותה רשת WiFi
- המחשב והטלפון חייבים להיות באותו WiFi

**פתרון 3:** נסה localhost במחשב קודם
- פתח `http://localhost:9800` במחשב כדי לוודא שהשרת עובד

### 🔴 השרת לא מתחיל

**פתרון:** בדוק אם הפורט תפוס
```powershell
# בדוק מי משתמש בפורט:
netstat -ano | findstr "9800"

# סגור את התהליך (אם צריך):
taskkill /F /PID <מספר-תהליך>
```

### 🔴 Cursor לא נפתח

**פתרון:** ערוך את קובץ `.env`
1. פתח את הקובץ `.env` בנוטפד
2. בדוק שהשורה `V2_CURSOR_EXE=...` מצביעה למקום הנכון
3. שמור ונסה שוב

### 🔴 שכחתי את הסיסמה

**פתרון:**
1. פתח את הקובץ `.env` בנוטפד
2. שנה את השורה `V2_PASSWORD=...` לסיסמה חדשה
3. שמור והפעל מחדש את השרת
4. כל המכשירים יצטרכו להתחבר מחדש
