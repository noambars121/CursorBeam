# 🧪 **בדיקת התכונות החדשות**

## ✅ **רשימת בדיקות:**

### 1. **בדיקת Streaming בזמן אמת** ⚡

```
נסה:
1. שלח הודעה מהטלפון: "תסביר לי מה זה React hooks"
2. בדוק שהתשובה מגיעה מילה במילה בזמן אמת
3. שים לב לconsole - אמור לראות: "🚀 Fast polling started"
4. אחרי שהתשובה הסתיימה - אמור לראות: "✅ Fast polling stopped"

תוצאה צפויה:
✓ תראה את המילים מופיעות אחת אחרי השנייה (כל 300ms)
✓ אחרי שהתשובה נגמרת, polling חוזר ל-3 שניות
```

---

### 2. **בדיקת כפתור ביטול** ❌

```
נסה:
1. שלח הודעה מהטלפון: "בדיקה 123"
2. שים לב לכפתור ❌ שמופיע
3. לחץ עליו תוך 10 שניות
4. ההודעה אמורה להיעלם

תוצאה צפויה:
✓ כפתור ❌ מופיע מיד אחרי שליחה
✓ לחיצה מוחקת את ההודעה
✓ Toast: "ההודעה בוטלה"
✓ הכפתור נעלם אחרי 10 שניות
```

---

### 3. **בדיקת העלאת תמונה** 📷

```
נסה:
1. לחץ על כפתור 📷
2. בחר תמונה מהגלריה
3. תצוגה מקדימה אמורה להופיע
4. לחץ ✕ להסרת התמונה
5. נסה שוב להעלות תמונה
6. שלח הודעה עם תמונה

תוצאה צפויה:
✓ תצוגה מקדימה של התמונה
✓ Toast: "תמונה נטענה ✓"
✓ לחיצה על ✕ מסירה את התמונה
✓ Toast: "תמונה הוסרה"
✓ שליחה עם תמונה - הטקסט נשלח, תמונה מתעלמים (לא נתמך עדיין)
```

---

### 4. **בדיקת בחירת מודל** 🤖

```
נסה:
1. לך לטאב Chat
2. פתח dropdown "מודל"
3. בחר מודלים שונים:
   - Claude 4.5 Sonnet
   - GPT-5 Codex
   - Gemini 2.5 Pro
4. רענן את הדף
5. המודל אמור להישאר

תוצאה צפויה:
✓ רואים את כל המודלים ברשימה
✓ בחירה שומרת ב-localStorage
✓ אחרי refresh - המודל נשאר זהה
```

---

### 5. **בדיקת בחירת פרויקט** 📁

```
נסה:
1. לך לטאב Chat
2. פתח dropdown "פרויקט"
3. בחר פרויקט שונה
4. רענן את הדף
5. הפרויקט אמור להישאר

תוצאה צפויה:
✓ רואים את כל הפרויקטים
✓ פרויקט פעיל מסומן: "(פועל)"
✓ בחירה שומרת ב-localStorage
✓ אחרי refresh - הפרויקט נשאר זהה
```

---

### 6. **בדיקת הצ'אט הרגיל מוסתר** 👻

```
נסה:
1. לך לטאב Chat
2. גלול למטה

תוצאה צפויה:
✓ רק Remote Chat (UIA) נראה
✓ הצ'אט הרגיל מוסתר לגמרי
✓ אין divider (קו מפריד)
```

---

### 7. **בדיקת UIA Status** 🔌

```
נסה:
1. סגור את UIA Host (Ctrl+C)
2. בדוק את ה-badge למעלה בRemote Chat
3. הפעל שוב את UIA Host

תוצאה צפויה:
✓ UIA Host מחובר ✓ (ירוק) - כש-Host רץ
✓ UIA Host לא זמין (צהוב) - כש-Host לא רץ
```

---

### 8. **בדיקת Rate Limiting** 🛡️

```
נסה:
1. שלח 35 הודעות תוך דקה
2. ההודעה ה-31 אמורה להיכשל

תוצאה צפויה:
✓ 30 הודעות ראשונות עובר
✓ הודעה 31 - Toast אדום: "Rate limit exceeded"
✓ אחרי דקה - אפשר שוב לשלוח
```

---

### 9. **בדיקת Sanitization** 🧹

```
נסה:
1. שלח הודעה עם תווים מיוחדים
2. שלח הודעה ארוכה מדי (>4000 תווים)

תוצאה צפויה:
✓ תווים מיוחדים נמחקים/מנורמלים
✓ הודעה ארוכה - נחתכת ל-4000 תווים
```

---

### 10. **בדיקת שליחת הודעה (הכי חשוב!)** 🚀

```
נסה:
1. וודא ש-Cursor פתוח
2. וודא ש-UIA Host רץ
3. שלח הודעה מהטלפון: "בדיקה מהאייפון"
4. עבור ל-Cursor ובדוק את הצ'אט הימני

תוצאה צפויה:
✓ ההודעה מופיעה ב-Cursor Chat בדיוק כמו שכתבת
✓ Toast: "✅ נשלח ל-Cursor!"
✓ התשובה חוזרת מילה במילה בזמן אמת
✓ הכל עובד מושלם! 🎉
```

---

## 🐛 **אם משהו לא עובד:**

### **הודעה לא מגיעה ל-Cursor:**
```powershell
# 1. בדוק ש-UIA Host רץ:
Get-Process CursorChatHost

# 2. בדוק logs של UIA Host:
# אמור לראות: POST /type ... 200 OK

# 3. בדוק שCursor פתוח:
Get-Process Cursor

# 4. נסה שוב
```

### **Streaming לא עובד:**
```javascript
// פתח Console בדפדפן (F12)
// אמור לראות:
// 🚀 Fast polling started - תשובות בזמן אמת!
// ...
// ✅ Fast polling stopped - חזרה לpolling רגיל
```

### **תמונה לא עובדת:**
```
תמונות עדיין לא נתמכות ב-UIA Host!
רק הטקסט נשלח. תמונות יגיעו בגרסה הבאה.
```
